Arithmetic Operator
==

Arithmetic operators are used to perform arithmetic on numbers

Operator | Description
-------|-------
^ | Exponentiation (x ^ y)
\* | Multiplication (x * y)
/ | Division (x / y)
mod | Remainder (x mod y)
+ | Addition (x + y)
- | Subtraction (x -y)

#### Example
```javascript
let x: int = 4
let y: int = 2

x ^ y // 16

x * y // 8

x / y // 2

x mod y // 0

x + y // 6

x - y // 2
```

Comparison Operators
==

Comparison operators are used in logical statements to determine difference or equality between two values

Operator | Description
------- | -------
== | Equals to
!= | Not equals to
< | Less than
\> | Greater than
=<  | Less than or equals to
\>= | Greater than or equals to

#### Example
```javascript
let x: int = 2
let y: int = 4

x == y // false

x != y // true

x < y // true

x > y // false

x =< y // true

x >= y // false
```


Logical Operators
==
Logical are used to determine the logic between two values

Operator | Description
------- | -------
! | Not 
&& | And
\|\| | Or 

#### Example
```javascript
let x: int = 2
let y: int = 4

(x < 5 && y > 1) // true

(x == 3 || y == 5) // false

!(x == y) // true
```
